# Queensland Crash Data Dashboard

##  Project Overview

This project presents an interactive Tableau dashboard analysing historical crash data across Queensland.

The goal of the analysis is to identify:

- Trends in crash frequency over time
- Geographic density of crashes
- Severity distribution
- Impact of speed limits
- Time-of-day crash patterns

---

##  Dataset

The dataset contains detailed crash records including:

- Crash severity
- Date and time
- Location coordinates
- Speed limit
- Road conditions
- Atmospheric conditions
- Casualty counts
- Vehicle types involved

Each record represents an individual crash event.

---

##  Dashboard Features

### 1️) Crash Trends Over Time
Line chart showing yearly crash counts.

### 2️) Geographic Density Map
Heatmap visualising crash hotspots across Queensland.

### 3️) Crash Severity by Time of Day
Bar chart analysing high-risk hours.

### 4️) Crash Severity by Speed Limit
Comparison of injury severity across speed categories.

### 5️) Regional Breakdown
Treemap highlighting crash distribution by region.
